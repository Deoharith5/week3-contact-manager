# Contact Management System

## Project Description

This project is a Contact Management System developed using Python. It allows users to store, search, update, delete, and manage contact information using dictionaries and JSON file storage.

## Features

- Add new contacts
- Search contacts with partial matching
- Update existing contacts
- Delete contacts
- Display all contacts
- Search by phone number
- Save contacts to JSON
- Load contacts automatically
- Export contacts to CSV
- Backup contacts
- View contact statistics
